01 · Guardrail Pipeline — Defense in Depth

A guardrailed app wraps the model in checks: input guardrails run before a token is spent, the model runs in the middle, and output guardrails run before the user sees anything. Cheap checks first, expensive ones last, and every decision logged. Pick a sample message (or type your own) and watch it flow through the stages — the first failing check short-circuits the rest.
