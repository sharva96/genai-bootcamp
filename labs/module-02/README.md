# Module 2 Labs — NLP, Transformers & LLM Fundamentals

Interactive HTML labs that visualize the technical foundations of language AI.
**No code required** — open any `.html` file directly in your browser.

## Labs in This Module

| Lab | File | Topic | What you'll learn |
|-----|------|-------|-------------------|
| Lab 1 | `lab01_nlp_tasks.html` | NLP Tasks Explorer | The full taxonomy of NLP — classification, NER, summarization, translation, Q&A, generation — with examples and real business uses |
| Lab 2 | `lab02_tokenization_playground.html` | Tokenization Playground | How text becomes numbers — BPE algorithm walkthrough, multi-language comparison, live token visualization, cost calculator |
| Lab 3 | `lab03_embeddings_visualizer.html` | Embeddings Visualizer | Words as vectors — semantic similarity, cosine similarity demo, the famous "king − man + woman ≈ queen" analogy |
| Lab 4 | `lab04_transformer_attention.html` | Transformer & Attention | The architecture behind every LLM — Q/K/V mechanism, multi-head attention, encoder vs decoder, with attention heatmaps |
| Lab 5 | `lab05_prompt_response_lifecycle.html` | Prompt-Response Lifecycle | The full pipeline from prompt to response — tokenization → embedding → attention → sampling, with sampling parameter demos |

## How to use

```bash
# Open any lab in your browser
open lab01_nlp_tasks.html       # macOS
xdg-open lab01_nlp_tasks.html   # Linux
start lab01_nlp_tasks.html      # Windows
```

Or just double-click the file.

## Learning Path

Complete the labs in order — each builds on concepts from the previous one.
Every lab ends with a 5-question self-check quiz with detailed explanations.
