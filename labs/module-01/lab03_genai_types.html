Text Generation

The most mature and widely deployed type. Models trained on trillions of tokens can now write, summarize, translate, answer questions, plan, and reason across virtually any topic. In 2025 we saw the rise of reasoning models that "think" before responding.

How it works
Decoder-only Transformer trained on next-token prediction. Reasoning models (o3, Claude with extended thinking) generate hidden "thinking" tokens before producing the visible response, dramatically improving accuracy on complex tasks.

Input → Output Example

Prompt (Input)

"Analyze this customer support ticket: 'My subscription auto-renewed even though I cancelled last month. I have a screenshot of the cancellation confirmation. Refund and confirm closure.' Classify urgency + sentiment, then draft a 60-word apology response."

⟶
Claude Sonnet 4.6
Generated Output

Urgency: High · Sentiment: Frustrated but evidence-based

Response: "Hi [Name], we're sorry — that auto-renewal shouldn't have happened given your cancellation. I've issued a full refund to your original payment method (3-5 business days) and confirmed your account is closed. Could you share the screenshot? It helps us fix the underlying bug. — Sarah, Support Team"

📋 More Prompts to Study

Use case: Documentation, knowledge transfer

Image Generation

Creates photorealistic or stylized images from text descriptions, or modifies, extends, and combines existing images. By 2026, professional-grade output is standard. The big shift: image generation built directly into chat models (GPT-5, Gemini 2.5) — no separate tool needed.

How it works
Diffusion models start from pure random noise and iteratively "denoise" toward a coherent image guided by text embeddings. FLUX.1 introduced rectified flow matching (faster, sharper). Multimodal LLMs like GPT-5 use a unified architecture that handles text and pixels in one model.

Input → Output Example

Text Prompt (Input)

"A photorealistic product hero shot of cordless headphones in matte black on a marble surface. Soft warm rim light from the right, slight reflection on marble, shallow depth of field. Shot on Hasselblad H6D, 80mm lens, commercial photography style. 16:9 aspect."

⟶
FLUX.1.1 / Midjourney v7
Generated Output

📸 A photorealistic product image is generated matching every specification — lighting direction, surface material, lens characteristics, depth-of-field falloff, and aspect ratio. Generated in 3-8 seconds. No photographer, studio, model, or post-processing required.

Use case: Website hero image, landing page, B2B marketing

Use case: Product website illustrations, blog graphics

Audio Generation

Synthesizes speech, music, and sound effects from text or audio inputs. The 2024-2025 leap: real-time voice conversations (GPT-5 Voice, Gemini Live) and music generation indistinguishable from human-produced tracks (Suno v4, Udio v2).

How it works
Autoregressive models predict audio tokens at high frequency; diffusion models denoise audio spectrograms. Voice cloning extracts a speaker embedding from a 5-30 second sample and conditions speech synthesis on it. Modern systems run end-to-end (text → audio waveform) without intermediate text-to-spectrogram steps.

Input → Output Example

Input

Text: "Welcome to TechCorp. Your call is important to us. Press 1 for sales, 2 for support, or stay on the line for an agent."

Voice style: British female, warm professional tone, pacing 150 WPM, slight pause after each option

⟶
ElevenLabs v3
Generated Output

🔊 A studio-quality audio file plays with the specified British accent, professional tone, natural pacing, and pauses where required. Indistinguishable from a professional voice actor. Generated in <1 second. Cost: ~$0.30/1K characters.

📋 More Prompts to Study

Audio prompts specify voice, emotion, pacing, and context — not just text.

🎙 Podcast intro
"Voice: warm American male, mid-30s, podcast host style. Energetic but conversational. Slight smile in voice.
Text: 'Hey everyone, welcome back to Founders Friday. I'm Mike, and today we're diving into the wild story of how a 23-year-old built a billion-dollar company in 18 months. Let's get into it.'"

Text: 'Thank you for calling Riverbank Insurance. To file a new claim, press 1. For an existing claim, press 2. For roadside assistance, press 3. To speak with an agent, press 0 at any time.'"

Use case: Customer-service IVR, automated phone systems

🎵 Music (Suno v4)
"An upbeat indie pop track, 120 BPM, sunny vibe. Acoustic guitar lead, light electronic percussion, layered vocal harmonies on the chorus. Lyrics about a road trip with friends, theme: freedom and youth. 90 seconds, radio edit structure."

Use case: Background music for videos, social content, ads (royalty-free)

Video Generation

The fastest-evolving type. Creates short to medium-length video clips from text, image, or video prompts. Sora 2 (OpenAI, late 2025) and Veo 3 (Google, 2025) crossed key quality thresholds: physically plausible motion, native audio generation, multi-shot scenes.

How it works
Diffusion Transformers applied across spatial (image) and temporal (time) dimensions. The model learns motion dynamics, physics, and scene coherence from millions of video clips. Sora uses "video as spacetime patches"; Veo 3 generates synchronized audio in the same pass. Outputs typically range from 5-60 seconds at 1080p-4K.

Input → Output Example

Text Prompt (Input)

"A cinematic 15-second drone shot descending toward a single house at dusk in a mountain valley. Smoke from chimney drifts up, soft golden-hour light, mist on the river below, gentle camera glide forward. Cinematic 24fps, anamorphic lens flare, warm color grade."

⟶
Sora 2 / Veo 3
Generated Output

🎥 A 15-second photorealistic video clip is generated with coherent motion, correct lighting transitions, realistic atmospheric particles (smoke, mist), and cinematic camera movement. Veo 3 includes synthesized ambient audio (wind, distant water). Cost: ~$0.50-2.00 per clip. Rendering: 1-5 minutes.

📋 More Prompts to Study

Video prompts specify subject, action, camera movement, lighting, style, and timing.

📣 Product ad

Use case: Product launch videos, social media ads

📚 Educational

Clean infographic style, soft pastel colors, gentle camera zoom on key moments. With voiceover-friendly pacing."

Use case: E-learning, scientific visualization, YouTube education

🏃 B-roll
"20 seconds of cinematic b-roll: morning coffee shop scene. Steam rising from a freshly poured latte, barista pouring milk into a cup with focused expression, customer scrolling phone at counter, warm morning light through windows. Multiple short cuts (2-3 seconds each), shallow depth of field, handheld feel."

Code Generation

The highest-ROI application of Generative AI. Writes, explains, debugs, refactors, and reviews code across all major languages. The 2024-2025 shift: from autocomplete to autonomous agents — tools like Claude Code, Cursor Agent, and Devin can complete multi-file feature work with minimal supervision.

How it works
Same Transformer architecture as text models, pre-trained heavily on code (GitHub, Stack Overflow, docs). Modern coding agents combine an LLM with: (1) file system tools, (2) shell access, (3) test runners, (4) iterative self-correction. They plan, execute, observe, and revise until tests pass.

Input → Output Example

Prompt (Input)

"Refactor this Python function to use async/await, add type hints, and wrap network calls in a retry decorator with exponential backoff (3 attempts, 1s/2s/4s delays). Handle httpx.TimeoutException specifically. [paste function]"

⟶
Claude Opus 4.7
Generated Code

@retry(attempts=3, backoff=lambda i: 2**i) async def fetch_user(user_id: int) -> User: try: async with httpx.AsyncClient(timeout=5) as c: r = await c.get(f"/users/{user_id}") r.raise_for_status() return User(**r.json()) except httpx.TimeoutException: logger.warning(f"Timeout for {user_id}") raise

📋 More Prompts to Study

Modern coding prompts specify language, constraints, context, and validation.

🐍 Build a function
"Write a Python function `parse_log_lines` that takes a list of nginx access log strings and returns a list of dicts with keys: timestamp (datetime), ip, method, path, status_code (int), response_size (int). Handle malformed lines by skipping them (don't raise). Add a docstring and 3 test cases."

Use case: Data processing, log analysis tools

🗄 Database query
"Write a PostgreSQL query for these tables:
orders(id, customer_id, created_at, total, status)
customers(id, name, email, plan, created_at)

Task: Top 20 customers by 90-day revenue, with their plan, total orders, average order value, and days since their last order. Only include customers on 'pro' or 'enterprise' plans with ≥3 orders."

Use case: Analytics, BI dashboards, customer insights

🔍 Explain code
"Explain what this React component does for someone unfamiliar with hooks. Walk through useState, useEffect, and the conditional rendering. Include a sequence of what happens from initial mount to user interaction. [paste component]"

Use case: Onboarding, code review prep, learning

🤖 Agentic task
"Inside this repo: (1) Add a /healthz endpoint to api/server.py that returns 200 with build info; (2) Add a Pytest test for it in tests/test_health.py; (3) Update README.md with usage; (4) Run the tests and confirm they pass. Make a single commit with a clear message."

Use case: Claude Code, Cursor Agent, Devin — autonomous multi-file work
