Side-by-Side Comparison

Layer Core idea How it learns Output type Real examples
AI Machines that simulate human intelligence Rules written by humans (early) or data-driven (modern) Decisions, actions, answers Chess engine, voice assistants, GPS navigation
ML Systems that learn patterns from data without explicit rules Supervised, unsupervised, or reinforcement signals Predictions, classifications Spam filter, fraud detection, Netflix recommendations
Deep Learning Multi-layered neural networks learning hierarchical features Backpropagation through many layers; needs large data + GPU Labels, embeddings, features Face recognition, speech-to-text, medical imaging
Generative AI Models that create new content resembling training data Pre-trained on massive corpora; aligned with RLHF Text, images, audio, video, code ChatGPT, DALL-E, Claude, Sora, GitHub Copilot
