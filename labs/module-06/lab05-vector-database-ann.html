05 · Vector DB & ANN Search

A vector database finds nearest neighbors fast. Brute force compares the query (cyan ✦) to every point — accurate but slow. ANN (e.g. HNSW) walks a navigation graph, hopping greedily toward the query and touching only a handful of points. Watch the comparison counts.
