# Accessories

## Bands
Replacement bands for the TimeFlow X1 come in silicone, woven nylon, leather,
and metal link. All bands use the same 22 mm quick-release pin, so they are
interchangeable without tools.

## Chargers
The X1 uses a proprietary magnetic charging dock. A spare dock and a travel
charging cable are sold separately. The watch is not compatible with standard
wireless charging pads.

## Screen protectors
TimeFlow sells tempered-glass screen protectors cut for the 1.4-inch display.
The AMOLED panel already has an oleophobic coating, so a protector is optional.

## Compatibility note
Accessories for the older TimeFlow V2 do not fit the X1 because the V2 used a
20 mm band and a different charging connector.
